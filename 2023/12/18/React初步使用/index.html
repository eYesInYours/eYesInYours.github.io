React Api：useEffect、useState

上述出现的useEffect，接收两个参数，第一个参数为函数，第二个参数为数组：

当数组为空时，可将useEffect看作组件挂载生命周期；

数组不为空时，比如是一个字段，那么当这个字段的数据变化时，会执行useEffect一个参数函数，相当于更新生命周期。
数组可传多值。

在useEffect第一个参数函数中return一个函数，即组件卸载调用——组件卸载生命周期。
